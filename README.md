# Healthcare Data Analytics Project – Patient Insights & Hospital Performance

---

## Project Overview

This project focuses on analyzing hospital patient data to understand patient flow, waiting time, and satisfaction levels. The goal is to derive actionable insights that can help improve hospital operations and patient experience.

The project follows an end-to-end data analytics workflow including data cleaning, exploratory data analysis (EDA), SQL-based analysis, and dashboard creation.

---

## Dataset Description

The dataset contains hospital patient data, including:

* Patient ID
* Visit Date
* Patient Wait Time
* Satisfaction Score
* Gender
* Attended Status

The dataset consists of over 9000 patient records and is used to analyze hospital performance metrics.

## Project Workflow

### 1. Data Cleaning (Python – Pandas)

* Cleaned raw Excel data (pivot format issues)
* Handled missing values
* Structured dataset for analysis
* Converted date and numeric fields

---

### 2. Exploratory Data Analysis (EDA)

* Analyzed daily patient trends
* Evaluated average waiting time
* Studied patient satisfaction distribution
* Identified patterns in patient visits

---

### 3. SQL Analysis

#### Basic Analysis

* Total number of patients
* Daily patient count
* Average waiting time

#### Intermediate Analysis

* Patient trends over time
* Average satisfaction score
* Gender-wise patient distribution

#### Advanced Analysis

* Peak days with highest patient load
* Relationship between wait time and satisfaction
* Attended vs non-attended analysis
* Trend-based insights for hospital efficiency

---

## Dashboard

A dashboard was created to visualize key metrics:

* Daily patient trends
* Average wait time
* Patient satisfaction score
* Gender distribution
* Attended status

---

## Key Insights

* Patient flow varies significantly across days
* Higher waiting time may impact satisfaction scores
* Certain days experience peak patient load
* Gender distribution remains relatively balanced

---

## Business Recommendations

* Optimize staff allocation during peak days
* Reduce waiting time to improve patient satisfaction
* Implement appointment scheduling improvements
* Monitor patient trends for better resource planning
